# 论文排版工具 - 界面升级说明

## 这次升级的目标

把它从“能用的工具界面”整理成更适合论文场景的 Windows 桌面效率工具：
- 更容易扫描参数
- 更容易找到主操作
- 更容易看日志和状态
- 更适合长时间填写和反复试跑

## 用户能直接感受到的变化

### 1. 顶部更简洁
- 标题区不再抢占主体空间
- 关键信息集中展示，打开后更快进入配置状态

### 2. 左侧导航更顺手
- 分类顺序更接近实际填写流程
- 导航区可以直接滚动，长列表查看更轻松
- 当前分组更容易识别

### 3. 主配置区更清楚
- 表单分组更明确
- 输入框、下拉框、增减按钮风格更统一
- 长页面可以直接滚动浏览，不容易迷失位置

### 4. 底部操作更集中
- 配置管理、文件选择、执行操作、运行日志聚合到底部区域
- 主操作按钮更突出，执行路径更直接

### 5. 日志反馈更清晰
- 日志区更容易阅读
- 运行过程、警告和结果更容易追踪

### 6. 整体风格更稳重
- 纯文字界面，减少干扰
- 更符合桌面工具而不是宣传页的使用预期
- 更适合论文这种长流程、高重复操作场景

## 当前使用方式

```bash
python run_gui.py
```

基本流程：
1. 在左侧导航中逐项检查参数
2. 在底部选择输入文件和输出文件
3. 视需要勾选“跳过目录生成”
4. 点击“开始格式化”
5. 在运行日志区查看进度和结果
