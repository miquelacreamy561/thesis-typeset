# 论文文档结构指南

默认结构基于华南农业大学 2024 规范。**不同学校要求不同，通过「特殊标题映射」自定义。**

## 完整结构示例

下方标注了工具会**检查**和**格式化**的所有内容。

```
摘要                                    ← [检查] 必须存在
这里是中文摘要正文...
关键词：城乡融合；乡村振兴；县域经济    ← [检查] 必须存在，格式「关键词：」

English Title of the Thesis             ← [检查] 关键词与Abstract之间需有英文标题
Author Name                             ← [格式化] 自动居中
（South China Agricultural University,   ← [检查] 需含University/College/China等
  Guangzhou 510642, China）
Abstract: English abstract here...       ← [检查] 必须存在，格式「Abstract:」
Key words: word1; word2; word3           ← [检查] 必须存在，格式「Key words:」

目录                                    ← [检查] 必须存在（内容自动生成）

第1章 绪论                              ← [检查] 至少一个「第X章」标题
1.1 研究背景                            ← [检查] 标题编号连续性（自动修正）
1.1.1 三级标题
1.1.1.1 四级标题
正文内容...城乡融合[1]...其他研究[2,3]   ← [检查] 引用编号是否有对应文献
...数据表明[4-6]...                      ← [格式化] 编号→交叉引用域

第2章 文献综述
2.1 相关理论
正文中引用文献[7]...

第3章 研究方法
图1 研究区位图                          ← [检查] 有插图时需有图题，编号连续
表1 评价指标体系                        ← [检查] 有表格时需有表题，编号连续
注：数据来源于xxx                       ← [格式化] 表注自动识别

第4章 结果与分析
图2 各维度得分对比                      ← [检查] 图编号应为2（接上文）
表2 回归分析结果                        ← [检查] 表编号应为2（接上文）

第5章 结论与展望

参考文献                                ← [检查] 必须存在
[1] Friedmann J. Title[M]. Cambridge,    ← [检查] 需有类型标识[J]/[M]等
    1966.                                ← [检查] 需有年份
[2] 张三. 标题[J]. 期刊, 2020.          ← [检查] 编号连续性、无重复
[3] ...                                  ← [检查] 正文引用了但文末无条目→警告
                                         ← [检查] 文末有条目但正文未引用→警告

致谢                                    ← [检查] 必须存在
感谢导师...

附录A 补充数据                          ← [可选] 附录标题需含字母
```

## 特殊标题映射（自定义结构）

工具通过「特殊标题映射」识别非编号章节。**映射决定了检查哪些结构、怎样显示。**

默认映射（GUI「高级」选项卡可编辑）：

| 匹配 (match) | 显示 (display) | 对齐 |
|--------------|----------------|------|
| 摘要 | 摘　　　　要 | 居中 |
| 目录 | 目　　　　录 | 居中 |
| 参考文献 | 参　考　文　献 | 居中 |
| 致谢 | 致　　　　谢 | 居中 |

### 适配其他学校

直接增删映射条目：

```yaml
special_titles:
  # 增加一个结构
  - match: "结论与展望"
    display: "结  论  与  展  望"
    align: center

  # 学校用「参考资料」
  - match: "参考资料"
    display: "参  考  资  料"
    align: center

  # 不需要致谢就删掉对应条目
```

新增的特殊标题若需视为一级标题，同步添加到：

```yaml
sections:
  special_h1:
    - "参考文献"
    - "致谢"
    - "结论与展望"    # ← 新增
```

### 映射的作用

1. **结构检查** — 文档是否包含这些部分
2. **标题格式化** — 按 display 和 align 显示
3. **章节边界** — 定位参考文献范围、引用交叉检查等

## 检查项汇总

| 检查内容 | 触发条件 | 警告示例 |
|----------|----------|----------|
| 中文摘要标题 | 始终 | 「缺少中文摘要标题」 |
| 中文关键词 | 始终 | 「缺少中文关键词」 |
| 英文题目 | 始终 | 「英文摘要页缺少英文题目」 |
| 英文作者/单位 | 始终 | 「英文摘要页缺少作者英文名与单位信息」 |
| Abstract | 始终 | 「缺少英文摘要」 |
| Key words | 始终 | 「缺少英文关键词」 |
| 目录标题 | 始终 | 「缺少目录标题」 |
| 正文章节标题 | 始终 | 「未检测到正文章节标题」 |
| 参考文献标题 | 特殊标题映射中存在 | 「缺少参考文献标题」 |
| 致谢标题 | 特殊标题映射中存在 | 「缺少致谢标题」 |
| 标题样式 | 始终 | 「未检测到标题样式」 |
| 标题编号连续性 | renumber_headings 开启 | 自动修正并提示 |
| 图题 | 文档含插图时 | 「检测到插图但缺少图题」 |
| 表题 | 文档含表格时 | 「检测到表格但缺少表题」 |
| 图表编号连续性 | check_numbering 开启 | 「正文图编号不连续」 |
| 文献类型标识 | 始终 | 「参考文献缺少类型标识[J]/[M]」 |
| 文献年份 | 始终 | 「参考文献缺少年份」 |
| 引用↔文献交叉匹配 | 始终 | 「正文引用了但文末无对应条目: [8]」 |
| 引用编号顺序 | 始终 | 「正文引用编号未按首次出现顺序排列」 |
| 文献编号连续性 | 始终 | 「参考文献编号不连续，缺少: [5]」 |
